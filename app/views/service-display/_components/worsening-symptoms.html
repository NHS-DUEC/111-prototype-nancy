What to do if you start to feel worse
{% if acuity == "high" %}
  1. Call 111
  2. Tell them 111 online said you needed to get help [timeframe].
  3. Tell them your symptoms have got worse and you need to speak to a nurse.

You might have to start the questions again, or answer new questions about any new symptoms.

{% else %}

Answer questions about your symptoms again.

If you have new symptoms, choose the one that’s worrying you most.

If you’re not sure how to choose, call 111.

{% endif %}
